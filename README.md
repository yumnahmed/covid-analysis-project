# 📊 COVID-19 Data Analysis Project

## 📌 Overview

This project focuses on analyzing global COVID-19 data to uncover trends, patterns, and key insights related to the spread of the virus, mortality rates, and recovery performance across different countries.

The analysis is performed using Python and presented through a Jupyter Notebook with clear visualizations and data-driven insights.


## 🎯 Objectives

* Understand the distribution of COVID-19 cases worldwide
* Analyze death and recovery rates
* Identify the most affected countries
* Provide meaningful insights to support decision-making


## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook


## 📂 Dataset

The dataset used in this project contains global COVID-19 statistics, including:

* Total Cases
* Total Deaths
* Total Recovered
* Active Cases
* Death Rate


## 🧹 Data Cleaning

The following steps were applied:

* Handling missing values using mean, median, and mode
* Removing duplicates
* Fixing data types
* Filtering inconsistent or noisy data


## 📊 Exploratory Data Analysis (EDA)

Key analyses performed:

* Distribution of death rates using histograms
* Total cases by continent
* Top 10 countries by confirmed cases
* Correlation between cases, deaths, and recoveries


## 📈 Key Insights

* Countries with higher case counts tend to have higher death counts
* Death rate varies significantly between regions
* Some countries show high recovery rates despite large case numbers


## 💡 Recommendations

* Focus on improving healthcare capacity in high-risk regions
* Increase awareness and preventive measures
* Monitor countries with rising trends closely
* Enhance data reporting accuracy
